# ЭзоДвиж · Кабинет ученика

Личный кабинет для учеников курса ЭзоДвиж. Статичный сайт (GitHub Pages), данные хранятся в браузере ученика (localStorage), регистрация по логину/паролю.

## Что внутри
- **Дашборд** — «Карта дня» (случайное напутствие / что запостить), шкала прогресса по темам, счётчик и дата следующего созвона.
- **Тесты** — проверка усвоения: выбор более триггерного заголовка, «сколько триггеров нашла», «где язык результата», открытые задания на подболи с эталоном.
- **Опросник тем** — что понятно / так себе / не понятно по каждой теме.
- **Мои продажи** — сколько заработано с начала проекта, залетевшие рилсы (смысл + охваты).
- **Рабочая тетрадь** — заметки для идей + таблица контент-плана (тема · цель · формат · эмоция · триггеры · дедлайн).
- **Домашка** — отправка работы преподавателю.
- **Настройки** — имя, созвоны, webhook Telegram-бота, бэкап данных.

## Дизайн
Минималистично и эзотерично: чёрный / белый / фиолетовый.

## Отправка домашки в Telegram
GitHub Pages — статика без сервера, поэтому автоотправка домашки в ТГ-бот включается только с бэкендом.
В **Настройках** есть поле `webhook-URL`: когда бот/сервер будут готовы, туда вписывается адрес приёма, и домашки начнут уходить в Telegram (POST JSON: `{student, theme, text}`). Пока бота нет — работа сохраняется локально и её можно скачать.

## Локальный запуск
Просто открыть `index.html` в браузере.
